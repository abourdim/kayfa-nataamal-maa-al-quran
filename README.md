# 📖 How to Deal with the Quran — كيف نتعامل مع القرآن

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **كيف نتعامل مع القرآن (How to Deal with the Quran)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on engaging with the Quran, 8 reflections on conscious reading, daily Quranic habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Floating particles background, glassmorphism panels
- Sound effects (Web Audio API)
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
